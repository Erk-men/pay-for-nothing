<decisions_made>

- Project: "Pay for Nothing" leaderboard — users pay simulated money for nothing, ranked by total donated
- Architecture: Layered (Route → Service → Model → DB). Business logic in services only.
- Frontend: Vanilla JS SPA with hash-based routing (#leaderboard, #auth, #dashboard)
- Database: SQLite via better-sqlite3 (synchronous, no install needed)
- Auth: JWT tokens (24h expiry) + bcryptjs password hashing
- Security: All 7 layers — bcrypt, JWT expiry, rate limiting, helmet, parameterized queries, CORS, .env
- Testing: Jest with --runInBand, in-memory SQLite (:memory:) for tests
- PDF: pdfkit, streamed directly to HTTP response (no temp file)
- Swagger: Static swagger.json served via swagger-ui-express at /api-docs
- All code committed to GitHub (required by course)
